# Mpuza_Dashboard_ReactJs-TailwindCss
A modern **job and social dashboard UI** built with **React.js** and **Tailwind CSS**.  
This project simulates a platform where users can post jobs, view opportunities, and interact with content.

## ✨ Features

- 👤 User profile section  
- 📝 Create post UI  
- 💼 Job listing cards  
- ⏳ Countdown timer display  
- ❤️ Like, comment, share buttons  
- 📊 Company and analytics section  
- 📱 Fully responsive design  
- 🎨 Styled with Tailwind CSS

## 🛠️ Tech Stack

- React.js

- JavaScript
